### 4.1 概述
主要功能是在数传SPP+BLE的基础上，增加了由上位机或其他MCU可以通过UART对接蓝牙芯片进行基本的配置、状态获取、控制连接断开以及数据收发等操作。
定义一套串口的控制协议，具体请查看协议文档《蓝牙AT协议》。
